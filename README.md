# Zaya's Ananda — Full-stack вэб платформ

Далд ухамсар, энерги-мэдээллийн судалгааны төвийн **бүрэн ажиллагаатай** вэбсайт.
Next.js (App Router) + Tailwind CSS-ээр бүтээгдсэн бөгөөд үйлчилгээ, онлайн сургалт,
дэлгүүр, сагс, захиалга, хэрэглэгчийн бүртгэл (нэвтрэх/бүртгүүлэх) зэрэг бүх үндсэн
функцийг агуулна.

> Анхаарах: контент (үйлчилгээ, сургалт, бүтээгдэхүүний нэр, үнэ, тайлбар) нь жишиг
> зориулалтаар бичигдсэн. `data/content.ts` файлаас өөрийн бодит мэдээллээ амархан солих
> боломжтой.

---

## ✨ Онцлогууд

- **Нүүр хуудас** — баннер, онцлох үйлчилгээ/сургалт/бүтээгдэхүүн, сэтгэгдэл, дуудлага.
- **Үйлчилгээ** — жагсаалт + дэлгэрэнгүй хуудас бүрд тус тусдаа.
- **Онлайн сургалт** — түвшин, хичээлийн тоо, үр дүнгийн жагсаалттай.
- **Дэлгүүр** — бүтээгдэхүүн, хямдрал, нөөцийн төлөв.
- **Сагс** — localStorage-д хадгалагдах, хажуугийн drawer + бүрэн хуудас.
- **Захиалга (checkout)** — серверт захиалга үүсгэх, баталгаажуулалт.
- **Хэрэглэгч** — бүртгүүлэх / нэвтрэх / гарах (JWT, httpOnly cookie).
- **Миний булан** — профайл + захиалгын түүх.
- **Холбоо барих** — серверт хадгалагдах зурвасын форм.
- Бүрэн responsive, монгол хэл дээр, орчин үеийн UI/UX.

## 🌐 Олон хэл (5 хэл)

Сайт **Монгол, Англи (English), Солонгос (한국어), Япон (日本語), Хятад (中文)** хэлийг бүрэн дэмжих ба
толгой хэсгийн хэл солих товчоор **шууд, refresh хийлгүй** солигдоно. Сонгосон хэл хөтөч дотор хадгалагдана.

- UI бичвэрүүд: `data/strings.ts`
- Контент (үйлчилгээ, сургалт, бүтээгдэхүүн, бидний тухай): `data/content.ts` — талбар бүр `{ mn, en, ko, ja, zh }` хэлбэртэй
- Захиалга үүсгэхэд барааны нэр сонгосон хэлээр серверт хадгалагдана
- Шинэ хэл нэмэх: `lib/types.ts` доторх `Locale`-д код нэмж, `lib/i18n-core.ts`-д жагсаалтад нэмээд орчуулгуудаа бичнэ
- Солонгос/Япон/Хятад үсгийг Noto Sans CJK фонтоор дэмжсэн

## 🧱 Технологи

| Давхарга | Технологи |
| --- | --- |
| Frontend | Next.js 14 (App Router), React 18, TypeScript |
| Загвар | Tailwind CSS (өөрийн дизайн систем) |

## 🚀 Суулгах ба ажиллуулах

```bash
# 1. Хамаарал суулгах
npm install

# 2. Орчны хувьсагч (заавал биш, гэхдээ санал болгоно)
cp .env.example .env.local
#   .env.local доторх AUTH_SECRET-ийг өөрийн нууц утгаар солино уу

# 3. Хөгжүүлэлтийн серверийг асаах
npm run dev
#   http://localhost:3000 хаягаар нээнэ

# 4. Продакшн build + асаах
npm run build
npm start
```

## ✏️ Контент засах

Бүх үйлчилгээ, сургалт, бүтээгдэхүүн, баг, сэтгэгдэл, түгээмэл асуултыг
**`data/content.ts`** файлаас засна. Зураг ашиглахгүйгээр өнгөт градиент + бэлгэдэл
(glyph) ашигласан тул `glyph` ба `tone` талбараар хялбар тохируулна.

`tone` боломжит утга: `violet`, `gold`, `jade`, `rose`, `sky`.

## 🗄️ Бодит өгөгдлийн сан руу шилжих

Одоогийн хувилбар нь энгийн JSON файл-санг (`data/db.json`) ашигладаг тул нэмэлт
тохиргоогүйгээр шууд ажиллана. Бодит SQL/Postgres руу шилжихдээ зөвхөн **`lib/db.ts`**
болон **`lib/repo.ts`** доторх функцуудыг (Prisma эсвэл өөр ORM-оор) солиход хангалттай —
бусад код өөрчлөгдөхгүй.

## 🔐 Аюулгүй байдлын тэмдэглэл

- Продакшнд `AUTH_SECRET`-ийг заавал хүчтэй утгаар солино уу.
- Нууц үг `scrypt`-ээр хеш хийгдэж хадгалагдана (цэвэр текстээр хадгалахгүй).
- Захиалгын үнэ серверт дахин шалгагдана (client-ийн өөрчлөлтөөс хамгаална).

## 📌 Анхаарах зүйлс

- Энэ нь **жишиг (demo)** систем тул бодит төлбөрийн систем холбогдоогүй.
  Захиалга нь өгөгдлийн санд хадгалагдах ба "Баталгаажсан" төлөвтэй болно.
- JSON файл-сан нь нэг серверийн орчинд тохиромжтой. Олон сервер / serverless
  орчинд бол дээрх бодит өгөгдлийн санг ашиглана уу.
